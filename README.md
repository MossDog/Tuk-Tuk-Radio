
# Tuk Tuk Radio — Band Website & Portfolio

This is a promotional website for the band **Tuk Tuk Radio**, built as a showcase of my web development skills and project structure on GitHub. It is not intended for distribution as a generic app or template.

## About the Project

- **Purpose:** Promote Tuk Tuk Radio, provide info about the band, and demonstrate modern web development best practices.
- **Features:**
  - Band bio and member profiles
  - Spotify-powered music tracks
  - Media gallery (photos & videos)
  - Responsive, accessible, and visually appealing UI
  - Social links (Instagram, Spotify)

## Tech Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS 4 & DaisyUI
- Lucide Icons

## Project Structure

- `app/` — Main Next.js app directory
- `components/` — UI components (TopBar, Gallery, Tracks, etc.)
- `lib/` — Utility and API logic (Spotify integration, gallery data)
- `public/` — Static assets (images, gallery media)
- `types/` — TypeScript types
